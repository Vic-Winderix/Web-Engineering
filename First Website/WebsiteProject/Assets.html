Animaties

We zijn begonnen bij het maken van onze animaties. We wouden verschillende animaties gebruiken voor ons karakter, bv lopen, sterven, stil staan, om zo onze game dynamischer te maken. Voor de animaties gebruiken we een gewone afbeelding waarom ons karakter in verschillende poses achter elkaar staat. Door deze afbeelding vervolgens in frames op te delen en achter elkaar af te spelen kunnen we het personage tot leven wekken. De tools die Godot ons aanrijkt zijn zeer handig om dit te doen. Door te spelen met de snelheid van de frames kan je je animaties aanpassen zodat deze in je game past.

Map

Voor het maken van onze map maken we gebruik van een tileset. Deze werkt ongeveer hetzelfde als het maken van een animatie. Je hebt een grote afbeelding waarop je een raster kan maken. Dit raster zijn dan allemaal tegels die ge kan gebruiken om je map vorm te geven. Je selectereerd één of meerdere tegels en vervolgens leg je deze als het ware neer op je scherm. Om de fysieke effecten te doen loppen maken we gebruik van layers. Deze layers zorgen ervoor dat je karakter voor of achter bepaalde objecten wandeld. Als je dus wilt verdijwnen achter een muur zal je de layer van de muur boven de layer van de speler plaatsen.
